Compact square button for a single icon — toolbars, list-row actions, nav. Always pass a `label`.

```jsx
<IconButton label="Search"><i data-lucide="search" /></IconButton>
<IconButton label="Add" variant="filled"><i data-lucide="plus" /></IconButton>
<IconButton label="Star" active><i data-lucide="star" /></IconButton>
```

Variants: `ghost` (default, transparent), `outline`, `filled` (blue). `active` tints it blue for toggled state.

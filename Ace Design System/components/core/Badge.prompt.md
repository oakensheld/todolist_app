Tinted pill for status, priority, counts, or category labels. Category tones match the planner color-coding.

```jsx
<Badge tone="success" dot>On track</Badge>
<Badge tone="interview">Interview</Badge>
<Badge tone="danger">Overdue</Badge>
<Badge tone="neutral" size="sm">3</Badge>
```

Tones: semantic (`neutral/primary/success/warning/danger`) + category (`study/project/interview/personal`).
